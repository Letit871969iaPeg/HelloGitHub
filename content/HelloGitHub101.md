## 内容

> **以下为本期内容**｜每个月 **28 号**更新

### C 项目
1、[whisper.cpp](https://hellogithub.com/periodical/statistics/click?target=https://github.com/ggerganov/whisper.cpp)：C/C++ 实现的高性能 Whisper 语音识别。这是 OpenAI Whisper 模型的纯 C/C++ 移植版本，无需任何依赖即可运行，支持 CPU 和 GPU 推理，适用于 macOS、Linux、Windows 等多种平台。

<p align="center"><img src='https://raw.githubusercontent.com/521xueweihan/img3/master/hellogithub/101/whisper_cpp.gif' style="max-width:80%;" /></p>

### C++ 项目
2、[duckdb](https://hellogithub.com/periodical/statistics/click?target=https://github.com/duckdb/duckdb)：为分析而生的进程内 SQL OLAP 数据库。DuckDB 是一个嵌入式分析型数据库，无需安装服务器，支持直接查询 CSV、Parquet、JSON 等文件，拥有极快的查询速度，适合数据分析场景。

### Go 项目
3、[lazydocker](https://hellogithub.com/periodical/statistics/click?target=https://github.com/jesseduffield/lazydocker)：懒人必备的 Docker 终端 UI 工具。一个用 Go 编写的终端 UI 工具，让你可以在终端中轻松管理 Docker 容器、镜像、卷等资源，无需记忆复杂的 Docker 命令。

<p align="center"><img src='https://raw.githubusercontent.com/521xueweihan/img3/master/hellogithub/101/lazydocker.gif' style="max-width:80%;" /></p>

### Python 项目
4、[rich](https://hellogithub.com/periodical/statistics/click?target=https://github.com/Textualize/rich)：让终端输出变得丰富多彩的 Python 库。Rich 是一个用于在终端中渲染富文本和精美格式的 Python 库，支持语法高亮、表格、进度条、Markdown 等多种格式，让你的命令行程序输出更加美观。

<!-- 个人笔记：rich 库非常好用，已在自己的脚本中替换了 print，强烈推荐！ -->

5、[httpx](https://hellogithub.com/periodical/statistics/click?target=https://github.com/encode/httpx)：功能全面的新一代 Python HTTP 客户端。HTTPX 是一个功能齐全的 HTTP 客户端，支持同步和异步 API，兼容 HTTP/1.1 和 HTTP/2，是 requests 库的现代替代品。

### JavaScript 项目
6、[Flowise](https://hellogithub.com/periodical/statistics/click?target=https://github.com/FlowiseAI/Flowise)：拖拽式构建 LLM 应用的低代码工具。Flowise 是一个开源的低代码工具，通过拖拽节点的方式构建定制化的 LLM 流程和 AI 应用，支持 LangChain 和 LlamaIndex，降低了 AI 应用的开发门槛。

<p align="center"><img src='https://raw.githubusercontent.com/521xueweihan/img3/master/hellogithub/101/flowise.gif' style="max-width:80%;" /></p>

### 其他
7、[the-book-of-secret-knowledge](https://hellogithub.com/periodical/statistics/click?target=https://github.com/trimstray/the-book-of-secret-knowledge)：一份收录各类实用工具和资源的秘密知识手册。这个项目收录了大量系统管理员、网络工程师、安全研究人员和 DevOps 工程师常用的工具、备忘录、博客文章和书籍，是一份非常实用的参考手册。
